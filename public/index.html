AI Assistant

Powered by clawdbot / OpenClaw

Quick Start

  1. Run npm run setup to configure
  2. Set your API key (Anthropic or OpenAI)
  3. Create a Telegram bot via @BotFather
  4. Run npm start to launch the gateway
  5. Message your bot on Telegram!

Gateway Dashboard

When the gateway is running, the clawdbot dashboard is available at:
http://localhost:18789/

Useful Commands

npm run setup - Interactive setup wizard
npm start - Start the gateway
npm start -- --verbose - Start with verbose logging
npm run status - Check installation status
