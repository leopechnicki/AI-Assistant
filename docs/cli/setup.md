# `hex setup`

Initialize `~/.hex/hex.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
hex setup
hex setup --workspace ~/.hex/workspace
```

To run the wizard via setup:

```bash
hex setup --wizard
```
